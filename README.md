# AsynDz4
У цих завданнях продемонстровано роботу з класом CompletableFuture для виконання асинхронних обчислень.

Завдання 1: Факторіал від суми елементів масивів
У цьому завданні:

Генерується масив із 10 випадкових цілих чисел (1–10).

Усі елементи масиву збільшуються на 5.

Обчислюється факторіал суми елементів початкового та зміненого масивів.


Завдання 2: Мінімальна сума пар чисел
У цьому завданні:

Генерується масив із 20 випадкових натуральних чисел (1–100).

Обчислюється мінімальна сума пар чисел у вигляді (a1 + a2, a2 + a3, ...).
